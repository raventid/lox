#+TITLE: Readme

|------------+------------------------------------------------------------------------|
| Name       | Expr                                                                   |
|------------+------------------------------------------------------------------------|
| expression | equality ;                                                             |
|------------+------------------------------------------------------------------------|
| equality   | comparison ( ( "!=" or "==" ) comparison )* ;                          |
|------------+------------------------------------------------------------------------|
| comparison | term ( ( ">" or ">=" or "<" or "<=" ) term )* ;                        |
|------------+------------------------------------------------------------------------|
| term       | factor ( ( "-" or "+" ) factor )* ;                                    |
|------------+------------------------------------------------------------------------|
| factor     | unary ( ( "/" or "*" ) unary )* ;                                      |
|------------+------------------------------------------------------------------------|
| unary      | ("!" or "-") unary or primary ;                                        |
|------------+------------------------------------------------------------------------|
| primary    | NUMBER or STRING or "true" or "false" or "nil" or "(" expression ")" ; |
|------------+------------------------------------------------------------------------|
